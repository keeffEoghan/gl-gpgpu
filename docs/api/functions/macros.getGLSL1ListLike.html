Function getGLSL1ListLike

  • getGLSL1ListLike(type: string, name: string, a: array<number, array<number>>, qualify??: string, init??: string): string

  • Generates an array-like declaration, as a GLSL1 syntax string.

    Workaround for lack of const arrays in GLSL < 3. Adds a lookup macro function; slow here, but standard.

    See

    getGLSLListBase

    Example

      getGLSL1ListLike('float', 'list', [0, 1, 2], 'const'); // =>
      'const int list_l = 3;'+cr+
      'const int list_0 = float(0);'+cr+
      'const int list_1 = float(1);'+cr+
      'const int list_2 = float(2);\n'+
      '// Index macro `list_i` (e.g: `list_i(0)`) may be slow, `+
        'use name (e.g: `list_0`) if possible.\n'+
      '#define list_i(i) ((i == 2)? list_2 : ((i == 1)? list_1 : list_0))\n';

    Returns

    The GLSL1 array-like declaration string.

    Parameters

    • type: string

      The GLSL list data-type.

    • name: string

      The name of the GLSL list variable.

    • a: array<number, array<number>>

      The list of GLSL values.

    • Optional qualify: string = ''

      A GLSL qualifier, if needed.

    • Optional init: string = type

      A data-type initialiser, type by default.
